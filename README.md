# Chicken-Market-Management-Java
A Java application for managing chicken market data, calculating prices, and checking market eligibility based on weight.
# Chicken Market Management System 🐔

Chicken Market Management System is a Java-based application that manages chicken information and determines whether chickens are ready for the market based on their weight.

## Project Description
The application creates chicken objects with an ID and weight, then checks if each chicken meets the required weight for selling. It also calculates the chicken price based on its weight and the daily price value.

The project demonstrates the use of Object-Oriented Programming concepts, including classes, objects, constructors, static variables, static methods, and instance methods.

## Features
- Create chicken objects with ID and weight.
- Check if a chicken is ready for the market.
- Calculate the selling price based on weight.
- Update daily price and required weight.
- Display chicken information and market status.

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Classes and Objects
- Static Variables and Methods

## How It Works
The program stores general market settings such as daily price and required weight using static variables. Each chicken object has its own ID and weight. The program compares the chicken weight with the required weight and calculates the price if it is ready for the market.

## Purpose
This project demonstrates fundamental Java programming concepts, including object creation, encapsulation, static members, and method implementation.
